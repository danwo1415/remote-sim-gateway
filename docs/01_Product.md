# Remote SIM Gateway Product Specification V1.0

## 定位

Remote SIM Gateway 是一个面向个人用户的极简远程 SIM 控制工具。

它不是远程控制 Android 手机，而是远程控制一张 SIM 卡。

所有数据保存在用户自己的 VPS，不依赖第三方云服务。

## 项目目标

SIM 卡插在一台 Android 手机上，用户可以在任何浏览器访问网页，实现：

- 接收短信
- 发送短信
- 接听电话
- 拨打电话

## 系统架构

```text
SIM Card
    │
Android App（SIM Gateway）
    │
HTTPS / WebSocket
    │
VPS
    │
Web UI
```

## 页面

共 6 个页面：

1. 登录
2. 首页
3. 接收短信
4. 发送短信
5. 接听电话
6. 拨打电话

## 首页

首页仅四个模块：

- 接收短信：显示未读数量
- 发送短信：不显示数量
- 接听电话：显示未接数量
- 拨打电话：不显示数量

## 接收短信

进入页面后，所有未读短信立即变为已读，首页未读数量清零。

短信列表不显示已读/未读状态，只展示：

- 时间
- 号码
- 正文

## 发送短信

输入：

- 手机号码
- 短信内容

点击发送即可。

## 接听电话

进入页面后，首页未接数量清零。

电话记录只有三种类型：

- 📞 接听
- 📤 拨出
- ❌ 未接

## 拨打电话

输入号码，点击拨打。通话中可以挂断。

## Feature Freeze

V1 不增加以下功能：

- 联系人同步
- 通讯录管理
- 文件传输
- 剪贴板同步
- 相册
- 定位
- 通知同步
- Android 远程控制
- 视频
- 多设备
- 多用户
- 群发短信
- 多 SIM 管理
- 插件系统
